STEP 1] Start MySQL Server from Docker Compose

docker-compose -f mysql-adminer.yaml up -d

STEP 2] Access Adminer Web UI and verify the connectivity to the server

http://localhost:8080

hostname: <mysql-container-ip>
username: user
password: password
database: school_db

STEP 3] Download and add mysql driver library to the Embedded Micro Integrator Server Configuration  Wizard's  Select libraries section by clicking the + icon.

mysql-connector-java-8.0.19.jar

STEP 4] Save and Verify the Datasource Connectivity

STEP 5] Run the application on Micro Integrator

STEP 6] Verify the service

GET http://localhost:8290/services/RESTDataService/student


POST http://localhost:8290/services/RESTDataService/student
 

{
  "_poststudent": {
    "name" : "Will Smith",
    "school": "Beverly Hills School",
    "grade": 10
  }
}


STEP 7] Stop MySQL server container

docker-compose -f mysql-adminer.yaml down
